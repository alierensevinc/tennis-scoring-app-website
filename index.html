🎾 Match Point

Match Point is a sophisticated React Native mobile application built with Expo, designed to provide a premium tennis/sports experience with dynamic theming and comprehensive features.

✨ Key Features

🎨 Dynamic Theming

  • Three distinct court themes (Grass, Clay, Hard)
  • Dynamic app icon switching
  • Custom UI elements per theme

📱 Platform Support

  • iOS 13.4+
  • Android 6.0+
  • iPad/Tablet optimization

📍 Location Features

  • Location tracking
  • Interactive maps
  • Permission handling

📸 Media Features

  • Image sharing
  • Camera roll integration
  • Screenshot capabilities

🛠️ Technical Stack

Core Technologies:

- React Native 0.74.1
- TypeScript
- Expo SDK 51
- Hermes JavaScript Engine

🚀 Development Scripts

# iOS Development
npm run build-ios              # Build iOS app
npm run clean-build-ios        # Clean and rebuild iOS
npm run run-ios               # Run on iOS simulator

# iOS Production
npm run eas-build-ios-simulator    # Simulator build
npm run eas-build-ios-development  # Development build
npm run eas-build-ios-preview      # Preview build
npm run eas-build-ios-production   # Production build

# Android Development
npm run android               # Run on Android device

🎨 Theme System

The app implements three distinct tennis court themes:

  • Grass Court: Green theme with custom styling
  • Clay Court: Orange/Red theme with custom styling
  • Hard Court: Blue theme with custom styling
